INSTALANDO BIBLIOTECAS KICAD

1. No menu superior, vá até Preferências
2. Selecione Gerencie as bibliotecas de símbolos ou Gerencia as bibliotecas de footprints
3. Em seguida, clique no símbolo de pasta para adicionar uma biblioteca já existente
4. Vá até as pastas presentes no diretório atual, como a pasta digikey-kicad-library-master
5. Selecione a pasta com os símbolos ou a pasta com os footprints que deseja adicionar
6. Se for adicionar símbolos, selecione os arquivos específicos que deseja e clique em Abrir. Se for adicionar footprints, basta selecionar a pasta como um todo.
7. Pronto!
